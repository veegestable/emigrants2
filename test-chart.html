Chart Debug Test

Check the browser console for debug output when you navigate to the Sex chart.

Open the main app and look at the browser console to see:

  • Firebase data loading
  • Chart data processing
  • Available years
  • Chart rendering

Steps to debug:

  1. Start the dev server: npm run dev
  2. Open the app in browser
  3. Navigate to the Sex/Gender chart
  4. Upload the Sex CSV file if no data shows
  5. Check browser console for debug messages

Expected behavior:

The Sex chart should show a pie chart with MALE and FEMALE distribution, either for a selected year or totals across all years.
